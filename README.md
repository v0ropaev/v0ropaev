### 🚀 Featured Projects
- **On-prem AI platform** _(private / enterprise)_ — architecture & delivery lead of a closed-contour RAG assistant over corporate documents (regulations, contracts, scans) + OCR + integrations. Local LLM serving (**vLLM + Qwen 3**) behind a provider-pluggable **LiteLLM** gateway with deployment profiles (`local` / `hybrid` / `closed-contour`); hybrid retrieval (**Qdrant + OpenSearch + bge-m3 + reranker, RRF**) with **pre-retrieval ACL**; a **masking egress-filter** (Presidio + Natasha) so only pseudonymized text leaves the perimeter; **license-clean BOM** (SBOM + SPDX-gate) for boxed resale. Designed for FSTEC-K2 / 152-FZ.
- **[knowbase](https://github.com/v0ropaev/knowbase)** — Git-versioned, provenance-grounded knowledge layer extracted from a codebase, served to humans and AI agents via MCP.
- **[zta-internal-api-prototype](https://github.com/v0ropaev/zta-internal-api-prototype)** — Zero-Trust architecture for internal APIs: mutual TLS, JWT, RFC 8705 certificate-bound tokens; FastAPI reference with a central PDP, attack demos and benchmarks.
- **[enhanced-jwt-safeguards](https://github.com/v0ropaev/enhanced-jwt-safeguards)** — Secure authentication on FastAPI: JWT with rate limiting, revocation and refresh.
